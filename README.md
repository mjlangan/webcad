# WebCAD

This is very work-in-progress! Do not expect any kind of stability at the moment 😄

## What is this?

I use two CAD products a lot in my 3D printing work: TinkerCAD and FreeCAD. TinkerCAD is great for doing simple edits to existing STL files, but I wish it had some of the more "advanced" functionality of FreeCAD. This project seeks to strike a balance between the two and provide an approachable, web-based CAD tool for makers and hobbyist users who may not have a lot of prior exposure to CAD software.
